# Ativação e Escopo

- Use a skill somente para NestJS + DDD + Clean Architecture.
- Confirmação obrigatória ao ativar:
```
🏗️ NestJS Architect Skill Ativada
Padrões: DDD, Clean Architecture, Repository Pattern, Notification Pattern
```
- Ativa quando prompts mencionam: DDD, Clean Architecture, aggregate, use case, repository pattern, NestJS.
- Não ativa quando pedidos são "simples", "básico", JS puro ou sem Nest.
- Contexto mínimo: arquivos `.ts` e framework `nestjs|nest`.
