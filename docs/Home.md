# Home

Bem-vindo ao vault do projeto **coreto/summer26**.

## Acesso rápido

- [[00-Inbox/]] — capturas rápidas, sem organizar
- [[01-Notes/]] — notas permanentes
- [[99-Meta/templates/|Templates]] — modelos para novas notas

## Workflow (Zettelkasten-lite)

1. **Capture** tudo em `00-Inbox/` sem se preocupar com organização
2. **Processe** a inbox semanalmente → mova para `01-Notes/` com links `[[wikilinks]]`
3. **Conecte** notas relacionadas entre si

## Atalhos úteis

- `Cmd+O` — abrir nota por nome
- `Cmd+P` — paleta de comandos
- `Cmd+N` — nova nota
- `Cmd+Shift+K` — graph view
